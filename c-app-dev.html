APP 與開發者資源

透過手機 App 串接精準計量 IC 與感測裝置，提供即時監控與歷史紀錄， 並搭配技術文件與 SDK / API，協助工程師快速完成整合與開發。

一、APP 功能概述

App 是用戶與裝置之間的主要互動介面，除了即時量測與紀錄，也可作為 Demo 與展示工具。

裝置管理

支援藍牙 / Wi-Fi 連線裝置的新增、綁定與權限管理，適合多裝置情境或分層管理需求。

即時量測與歷史紀錄

以圖表與數字同步呈現量測數據，並支援匯出 CSV 供進一步分析或報表使用。

告警與通知

可設定門檻值，當量測數據異常時推送通知，協助使用者即時掌握設備狀態。

二、示範情境

以下示例展示如何結合 IC、模組與 App，快速驗證與展示不同領域的應用價值。

電池包健康監測

搭配電池包電芯級精準量測 IC，在 App 中查看 SOC / SOH、循環壽命與溫度分佈， 適用於電動車與儲能系統。

空間感測儀表

使用空間中個體物理特性量測 IC，顯示環境狀態、移動軌跡或安全監控資訊， 適用於智慧建築與 IoT 節點。

客製品牌 Demo App

提供 OEM / ODM 合作客戶專屬 Demo App 外觀與品牌元素，方便對外展示產品功能與價值。

系統 / 模組與 OEM / ODM 方案

以精準計量 IC 為核心，向上延伸至量測模組、系統板與整機設計，再結合 OEM / ODM 合作模式， 協助客戶從概念驗證（PoC）到量產導入。

一、系統 / 模組定位

系統 / 模組產品線以本公司的精準計量 IC 為核心，整合 MCU、通訊介面與感測器， 形成從模組到系統板的完整硬體平台。

量測模組

將 IC、必要的類比前端與通訊介面封裝為標準模組，預先完成校正與參數設定， 有助於縮短系統整合時程。

系統板 / 參考設計

提供針對特定應用的系統板，如電池包監測板、空間感測設備板，包含供電、保護電路、 感測器與通訊埠配置。

通訊與雲端連接

可搭配 Wi-Fi、BLE 或蜂巢式模組，並預留與雲端平台整合的能力，例如 MQTT 或 HTTP API。

二、方案類型：從模組到端到端

依據客戶既有資源與需求不同，可選擇不同程度的整合方案，從最精簡的 IC + 模組， 到完整的端到端解決方案。

方案一：IC + 模組整合

以既有精準計量 IC 與通訊模組為核心，提供標準化硬體與通訊協定， 適合希望快速導入量測功能的應用。

方案二：客製系統板

依產品外觀與功能需求，設計專屬系統板（MCU + Gauge + 通訊模組）， 支援多種感測器與電源架構。

方案三：端到端解決方案

結合雲端 / App / 裝置管理，提供從裝置註冊、資料上雲、帳號與授權管理的完整方案， 適合需要完整平台體驗的客戶。

三、OEM / ODM 服務內容

除了既有模組與系統板，速玥亦提供 OEM / ODM 形式的合作服務，以滿足不同客戶在產品開發、 系統導入與量產支援上的需求。

IC / 模組供應

針對已有系統架構的客戶，提供精準計量 IC 與模組，支援既有設計導入與替換， 降低重新設計成本。

系統與整機設計

從規格定義、電路設計到韌體開發與功能驗證，提供完整的系統與整機設計服務。

App 與平台整合

提供 App 功能規劃、UI 建議與 API 介接，能與現有平台或自有 App 方案整合。

四、合作與開發流程示意

以下流程說明從需求討論到量產導入與維運的各階段，可視實際專案彈性調整。

1. 需求討論

確認應用場景、量測項目、目標時程與預算範圍。

2. 方案評估與規格確認

選定適合的 IC / 模組與系統架構，並共同行程規格書與預估成本。

3. 設計與打樣

進行硬體、韌體與必要的 App / 平台整合開發，製作試產樣品。

4. 測試與修正

根據測試結果調整設計，優化量測精度、穩定度與使用體驗。

5. 量產導入

協助建立生產測試流程與品質管控項目，確保量產穩定。

6. 維運與升級

依產品生命週期提供技術支援與升級建議，延長產品價值。
